### ES6 문법 중 하나인 템플릿 리터럴 사용 방법
---

기존 ES5에서 문자열에서 변수를 사용하는 방식은 아래와 같다.
```js
var name = "홍길동";
alert("환영합니다! "+name+"님 ");
```

---

ES6에서 생긴 **템플릿 리터럴**을 사용하는 방식은 아래와 같다.
```js
let name = "홍길동"
alert(`환영합니다! ${name}님`);
```

문자열을 작성 시 따옴표가 아닌 백틱을 사용하면 문자열 사이에 ${변수명}을 이용하여 변수를 사용할 수 있다.
